## Working Agreement

- The Git repository is the project's single source of truth.
- Important decisions must be documented in the repository.
- Meeting notes will be stored in the meetings folder.
- Tasks will be tracked using repository issues or project boards.
- Team members are expected to update progress regularly.
- Critical project information should not exist only in chat messages or personal files.
